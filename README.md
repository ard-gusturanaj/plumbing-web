# LG Company Website

Plumbing company website built with React, TypeScript, Vite, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

## Build Scripts

```bash
npm run build
```

Creates an optimized production build in `dist/`.

```bash
npm run build:readable
```

Creates a non-minified build with sourcemaps in `dist/` so bundled files are easier to inspect.

## Important Note

Do not edit files inside `dist/` directly.  
All source changes should be made in `src/`, then rebuild.
